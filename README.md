# Timilehin Ogunsakin Portfolio

A premium, interactive portfolio built with React and Vite. The site uses a dark editorial aesthetic with gold and teal accents, layered motion, a custom cursor, scroll progress, and a subtle particle network effect.

## Features

- React + Vite setup
- Modular component-based structure
- Custom cursor and scroll progress indicator
- Scroll-aware navigation
- Hero particle network background
- Magnetic buttons and card hover interactions
- Responsive layout and reduced-motion support

## Tech Stack

- React
- Vite
- tsParticles
- GSAP

## Getting Started

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```
